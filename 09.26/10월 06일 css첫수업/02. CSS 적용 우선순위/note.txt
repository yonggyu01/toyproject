* CSS 적용 우선 순위

1. 요소 선택자로 선택하고 다른 효과를 작성한 경우
-- 모든 효과 적용


2. 요소 선택자로 선택하고 동일한 효과의 다른 값을 작성한 경우
-- 나중에 작성한 문장의 효과 적용


3. 다른 선택자로 동일한 요소를 선택하고 다른 효과를 작성한 경우
-- 모든 효과 적용


4. 다른 선택자로 동일한 요소를 선택하고 동일한 효과의 다른 값을 작성한 경우
-- 점수 환산법 활용

-- 요소 선택자 : 1
-- 클래스 선택자 : 10
-- 아이디 선택자 : 100
-- 인라인 스타일 : 1000

-- !important : 우선순위 무시하고 스타일 적용
