--javascript의 기본식, 연산자--

-기본식-

기본 키워드 및 JavaScript의 일반 식

  1. this

    2. this 키워드는 실행 문맥의 특별한 속성을 가리킵니다.

  2. funtion

    3. function 키워드는 함수를 정의합니다.

  3. class

    4. class 키워드는 클래스를 정의합니다.

  4. yield

    5. 생성기 함수를 일시정지 및 재개합니다.

  5. async fuction

    6. async function은 비동기 함수 표현식을 정의합니다.

  6. await

    7. 비동기 함수를 일시 중지했다가 다시 시작하고 promise의 resolution/rejection을 기다립니다.

  7. []

    8. 배열 초기자 및 리터럴 구문.


-좌변식-

좌변값은 할당 대상입니다

  1. new

    2. new 연산자는 생성자의 인스턴스를 만듭니다.

  2. new.target

    3. 생성자 문맥에서, new.target은 new에 의해 호출된 생성자를 말합니다.

  3. super

    4. super 키워드는 부모 생성자를 호출합니다.


-증가 및 감소-

접두/접미 증감 연산자입니다.

  1. A++

    2. 접미 증가 연산자.

  2. A--

    3. 접미 감소 연산자.

  3. ++A

    4. 접두 증가 연산자.

  4. --A

    5. 접두 감소 연산자.


-단항 연산자-

단항 연산은 피연산자가 하나뿐인 연산입니다.

  1. delete

    2. delete 연산자는 객체에서 속성을 지웁니다.

  2. void

    3. void 연산자는 식의 반환값을 버립니다.

  3. typeof

    4. typeof 연산자는 주어진 객체의 형을 판별합니다.

  4. +

    5. 단항 더하기 연산자는 피연산자를 숫자로 변환합니다.

  5. -

    6. 단항 부정 연산자는 피연산자를 숫자로 변환한 뒤 부호를 바꿉니다.

  6. !

    7. 논리 NOT 연산자.


-산술 연산자-

산술 연산자는 피연산자로 숫자 값(리터럴이나 변수)을 취하고 숫자 값 하나를 반환합니다.

  1. +

    2. 더하기 연산자.

  2. -

    3. 빼기 연산자.

  3. /

    4. 나누기 연산자.

  4. *

    5. 곱하기 연산자.

  5. %

    6. 나머지 연산자.

  6. **

    7. 거듭제곱 연산자.


-이진 비트 연산자-

비트 연산자는 피연산자를 32비트 집합(0과 1)으로 다루고 표준 JavaScript 숫자 값을 반환합니다.

  1. &

    2. 비트 AND.

  2. |

    3. 비트 OR.


-이진 논리 연산자-

논리 연산자는 보통 사용될 때 불리언(논리) 값으로 사용되고, 불리언 값을 반환합니다.

  1. &&

    2. 논리 AND.

  2. ||

    3. 논리 OR.

  3. ??

    4. Nullish 통합 연산자.


-삼항 연산자-

조건부 연산자는 조건의 논리값에 따라 두 값 중 하나를 반환합니다.


-할당 연산자-

할당 연산자는 값을 그 우변 피연산자의 값에 따라 좌변 피연산자에 할당합니다.

  1. =

    2. 할당 연산자.

  2. *=

    3. 곱셈 할당.

  3. **=

    4. 거듭제곱 할당.

  4. /=

    5. 나눗셈 할당.

  5. %=

    6. 나머지 할당.

  6. +=

    7. 덧셈 할당.

  7. -=

    8. 뺄셈 할당

  8. &=

    9. 비트 AND 할당.

  9. ^=

    10. 비트 XOR 할당.

  10. |=

    11. 비트 OR 할당.

  11. &&=

    12. 논리적 AND 할당.

  12. ||=

    13. 논리적 OR 할당.
