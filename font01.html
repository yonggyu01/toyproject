font의 다채로운 css 활용

font-family, font-size, color, font-weight, text-decoration을 사용하여 봅니다.

해당 여러 옵션들은 아래와 같습니다.

font-family: monospace 또는 sans-serif 등

font-size: 14px, 또는 원하는 OOpx 값을 넣습니다.

color: blue 또는 원하는 값을 넣습니다. brown, green 등...

font-weight: bold, 또는 lighter, normal 넣습니다.

text-decoration: underline, none, overline, line-through를 넣습니다.
